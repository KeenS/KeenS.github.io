Deprecating CIM

2017-01-29
  • Lisp
  • Common Lisp
  • CIM
I’m κeen. I have a sad news, I’ll stop the development of CIM and no more updates for new lisp impls will be provided. It has been months from when I stopped developping CIM and only providing updates for new lisp impl versions. From that point, CIM is a half-dead product. I know there are still someone using CIM though, so I kept updating. For present CIM users, I recomend to migrate to roswell, also a lisp installer and manager, which is well-maintained.
